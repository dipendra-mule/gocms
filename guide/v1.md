# CONTENT

### POST api/v1/post _protected_

```
{
  "title": "My First Post",
  "content": "This is my first post content with some markdown formatting."
  "author_id": 1
  "tags": ["tag1", "tag2"]
}
```

RESPONSE `201` CREATED

```
{
  "id": 1,
  "title": "My First Post",
  "author_id": 1,
  "created_at": "2023-01-01T00:00:00Z",
  "updated_at": "2023-01-01T00:00:00Z"
}
```

RESPONSE `400` BAD REQUEST

```
{
  "error": "invalid input"
}
```

### GET api/v1/post/:id _protected_

```
{
  "id": 1,
  "title": "My First Post",
  "author_id": 1,
  "created_at": "2023-01-01T00:00:00Z",
  "updated_at": "2023-01-01T00:00:00Z"
}
```

RESPONSE `404` NOT FOUND

```
{
  "error": "content not found"
}
```
